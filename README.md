GeoFSのUH60をUH60Jにする追加パーツ、リバリーが含まれます。
※使用にはリバリーセレクターアドオン、Extra vehicleアドオンが必要になります。

★使用方法★
1,main.jsの中のコードをすべてコピーし、Tampermonkey®に貼り付けてください。ブラウザのコンソールを使う場合はコードの「// ==/UserScript==」から下の(function()から始まる部分をすべてコピーして、コンソールに貼り付けてください。
2,「UH60Jbody_ocean.png」をダウンロードしてください。(UH60Jの本体のリバリーデータです。独自のものを使用する場合、追加パーツとの色の差が生まれることがあります。)
3,導入に成功すると下部タブに「UH60J」というタブが追加されます。
4,タブの中の「モデル表示」をクリックすると、追加パーツが追加されます。「洋上迷彩」ボタンで追加パーツ"のみ"リバリーが洋上迷彩になります。
5,リバリーセレクターの最下部の「lord external livery」の右の三角を開き、「direct」で先ほどの「UH60Jbody_ocean.png」をロードします。
完了です。
